# Agent Switchboard 0.2 Archive

These records describe the implemented task-first product through Phase 5 and
the investigations that led to it. Phase 6 replaces that product shape rather
than extending its public contracts.

The files remain useful as historical evidence for provider behavior,
transport acceptance, task/session semantics, and the clean-break exporter.
They are not current design commitments, are not packaged as active product
documentation, and must not be used to infer compatibility requirements for
the `0.3` registry, configuration, CLI, protocols, TUI, or DMS adapter.
